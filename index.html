CSS Grid Layout sample. Test the new CSS Grid layout and how to create grid based layouts and alignments. This is different from Flexbox which is for one-dimensional layouts in either colomns or rows. CSS Grids allow for two-dimensional layouts of rows and columns simultaneously with more complexity. We will look at properties for containers and items.
